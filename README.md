# Tied Column Design App

This Streamlit app assists with the design of short tied reinforced concrete columns based on ACI methodology.

## Features
- Estimates trial column size
- Computes eccentricity and γ
- Uses interaction diagram inputs
- Interpolates steel ratio ρg
- Selects reinforcement
- Designs ties and spacing
- Computes development and splice lengths

## How to Use
1. Enter Pu and Mu
2. Select column size
3. Input interaction diagram values (ρg at γ = 0.60 and 0.75)
4. View results

## Notes
- Based on ACI 318 concepts
- Intended for educational use
